# Project Documentation

This section contains project records that were moved out of the repository
root. Status, dashboard, phase, and summary pages are point-in-time records;
they are not substitutes for the current package, tests, or documentation
index.

## Sections

- `status/` - current implementation state and continuation status
- `reviews/` - code-review follow-up notes and summaries
- `summaries/` - executive summaries and consolidated project overviews
- `dashboard/` - dashboard-specific plans, reviews, and status docs
- `routers/` - router implementation summaries
- `phases/` - phased progress and completion reports
- `sdk-utilization/` - SDK utilization plans and implementation reports
- `migration/` - migration guidance and transition notes
